## 1. 项目简介

`dy-ks-mcp` 是一个给上层 AI Agent 使用的 MCP 工具层，服务于抖音/快手评论任务。

设计重点：

- Agent-first 编排
- 结构化错误与可接管执行
- 在不稳定页面环境下尽量“可恢复”而非“卡死”

## 2. 执行模式

- 抖音：半自动（浏览器 Agent 手动搜索与选目标，MCP 提交评论与校验）
- 快手：可自动（搜索、选目标、评论、校验）

## 3. 核心工具

- `check_login_status`
- `start_login`
- `search_posts`
- `prepare_comment_target`
- `submit_comment`
- `verify_comment`
- `run_comment_task`

## 4. 状态协议

统一返回：

- `success`
- `partial`
- `blocked`
- `failed`

错误结构：

- `error.stage`
- `error.code`
- `error.message`
- `error.retriable`
- `error.requires_agent`
- `error.agent_hints`
- `error.artifacts`

## 5. 防卡死保护

已内置：

- 请求级硬超时
- worker 执行超时
- relay 未附着快速失败（`RELAY_NOT_ATTACHED`）

## 6. 推荐闭环流程

1. `check_login_status`
2. 搜索与选目标（抖音手动、快手自动）
3. `submit_comment`
4. `verify_comment`

判定闭环：`submit_comment` 成功且 `verify_comment.exists=true`。
